# 20260429 PhD Experiences

本仓库用于整理一场 15 分钟中文经验分享 PPT，主题为：**分享博士国家奖学金获得的经验和心得**。

主要输出文件包括：

- [陆震-学习经验分享.pdf](output/陆震-学习经验分享.pdf)：演讲 PPT

## 内容范围

本分享面向公共卫生学院研究生群体，重点讨论国家奖学金相关的长期积累经验。内容包括：

1. 科研与学术
   - 如何快速进入科研状态，从入门到找到研究方向
   - 文献阅读方法：如何高效阅读、整理和形成自己的思路
   - 论文写作经验：选题、创新点、结构、投稿和返修
   - 调研 / 数据处理心得：避坑、效率提升和结果复盘
   - 如何和导师高效沟通：汇报节奏和问题表达

2. 时间管理与研究生生活平衡
   - 研一 / 研二 / 研三关键时间节点规划
   - 课程、科研、项目如何兼顾
   - 拖延、焦虑、压力大时的调整方法
   - 如何保持长期自律，避免内耗
   - 避免无效努力，少走弯路

3. 心态与成长
   - 科研受挫、实验失败、被拒稿时如何调整
   - 如何建立学术自信，不盲目内卷
   - 团队合作、师门相处和同门互助经验
